# پروژه شطرنج (Chess Game)
یک بازی شطرنج کامل نوشته شده به زبان C++ که شامل قوانین کامل شطرنج، رابط کاربری خط فرمان و منطق بازی کامل است.
## فهرست مطالب
- ویژگی‌ها
- پیش‌نیازها
- نحوه کامپایل و اجرا
- راهنمای بازی
- ساختار پروژه
- قوانین پیاده‌سازی شده
- نحوه ورودی گرفتن
- اشکال‌زدایی
## ویژگی‌ها
- تمام مهره‌های شطرنج: شاه، وزیر، رخ، فیل، اسب، سرباز
- حرکات ویژه:
  - قلعه (Castling کوتاه و بلند)
  - آن پاسان En Passant))
  - ترفیع سرباز Promotion))
- تشخیص وضعیت‌ها:
  - کیش Check))
  - کیش و مات (Checkmate)
  - پات (Stalemate)
- رابط کاربری خط فرمان با صفحه‌نمایش گرافیکی ساده
- سیستم Undo/Redo برای حرکات با استفاده از کلاس Move))
- ذخیره تاریخچه حرکات
## پیش‌نیازها
- کامپایلر( C++ GCC 7.0 یا بالاتر)
- سیستم عامل: Windows/Linux/MacOS
- حداقل حافظه: 128 MB RAM
## ️ نحوه کامپایل و اجرا
bash
# کامپایل تمام فایل‌ها

## راهنمای بازی
### نحوه ورودی گرفتن
ورودی‌ها به صورت استاندارد شطرنج با فرمت [خانه مبدأ] [خانه مقصد] وارد می‌شوند:
Enter the Position from which to move: e2
Enter the destination: e4
نکته: حروف کوچک و بزرگ هر دو قابل قبول هستند.

راهنما: 
- حروف بزرگ = مهره‌های سفید
- حروف کوچک = مهره‌های سیاه
- K = شاه، Q = وزیر، R = رخ، B = فیل، N = اسب، P = سرباز
### پیام‌های بازی
- CHECK! - شاه حریف در معرض کیش است
- CHECKMATE! - شاه حریف مات شد (پایان بازی)
- STALEMATE! - بازی مساوی شد (پات)
- Illegal Move! Your King would be in check. - حرکت غیرقانونی (شاه خودتان کیش می‌شود)
- Invalid Move! - حرکت غیرمجاز برای مهره
